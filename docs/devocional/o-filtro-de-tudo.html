O filtro de tudo

Versículo

"Todas as vossas coisas sejam feitas com amor."

1 Coríntios 16:14

Paulo resume um capítulo inteiro de instruções com uma frase curta: tudo, com amor. É quase um filtro para passar cada ação pela mesma pergunta. Não importa só o que se faz, importa o como.



Dá para fazer a coisa certa de um jeito frio, que machuca. Dá para ter razão e ainda assim ferir. Esse versículo lembra que o amor não é um detalhe a mais, é o modo que deveria atravessar tudo.



Hoje, antes de uma conversa, uma decisão ou uma resposta, vale passar pelo filtro: isso está sendo feito com amor? Não é sobre ser perfeito, é sobre deixar que o amor molde a forma das suas ações, mesmo nas pequenas.
