"Lâmpada para os meus pés é a tua palavra, e luz para o meu caminho."

Salmos 119:105

Uma lâmpada antiga iluminava só alguns passos na frente, não o caminho inteiro até o destino. Essa imagem é proposital: a Palavra de Deus normalmente não mostra o futuro distante, orienta a decisão de agora.



Isso explica por que muitas vezes não recebemos resposta para perguntas grandes, como o que vamos fazer nos próximos cinco anos, mas encontramos clareza suficiente para o próximo passo certo.



Se você está esperando ver o caminho todo antes de andar, talvez esteja pedindo mais luz do que precisa hoje. Pergunte: qual é o próximo passo visível, mesmo que o resto ainda esteja no escuro?
