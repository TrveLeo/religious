Onde o impossível esbarra

Versículo

"Para os homens é impossível, mas não para Deus, porque para Deus todas as coisas são possíveis."

Marcos 10:27

Jesus diz isso depois de uma afirmação dura, que deixou os discípulos perplexos. A reação deles foi quem pode então? A resposta não nega a dificuldade, ela reposiciona: o que é impossível para nós não é o limite de Deus.



Isso não é um cheque em branco para qualquer desejo. É um lembrete de que o cálculo humano de possibilidades não é a palavra final. Há coisas que travam na nossa lógica e se abrem na de Deus.



Hoje, se você olha para uma situação e só vê impossibilidade, vale segurar essa frase. O fim da sua capacidade não é necessariamente o fim das possibilidades. Onde a sua conta trava, a de Deus pode seguir.
