Suavidade que move

Versículo

"A palavra branda quebranta os ossos; o falar suave acalma a ira."

Provérbios 25:15

O provérbio aponta um paradoxo. A palavra suave, que parece fraca, tem força para quebrantar até o que é duro. Não é a gritaria que vence resistências, é a brandura persistente.



A gente costuma achar que firmeza é sinônimo de aspereza, que para ser levado a sério precisa endurecer o tom. Esse texto sugere o contrário: a suavidade, longe de ser fraqueza, costuma abrir portas que a dureza fecha.



Hoje, numa situação tensa, vale testar a palavra branda em vez do tom afiado. Não como manipulação, mas como sabedoria. Falar suave costuma alcançar o que o grito só endurece.
